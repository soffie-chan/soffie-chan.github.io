OVERVIEW
The Game Studies Handbook is a compliation of resources and advice for students. I found the handbook to be extremely useful but not interactive enough, which was an opinion various students held. I inquired from various faculty members of the Game Studies Department and recieved approval to develop the Game Studies Handbook into an interactive website to increase engagement from students.


ROLE
I spoke with faculty members regarding the Handbook and pursuing the website development as a project. This is a solo project - I will be coding and creating all assets!

THE PROBLEM

The Game Studies & Esports Department has a useful compilation of advice, resources, and information in the form of a Google Document. I found the handbook to be less interactive than desired, and therefore less engaging. Upon inquiry of Game Studies faculty, it was agreed upon that the Handbook could use better design choices in an interactive format.

THE SOLUTION

Students need an interactive, appealing way to use the Game Studies Handbook and book engagement. A website for the Handbook using customizable video-game themed assets can improve user interaction.
